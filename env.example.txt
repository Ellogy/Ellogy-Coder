# ============================================
# NOTES IMPORTANTES
# ============================================
#
# 1. Variables VITE_* : 
#    - Sont exposées côté client (dans le navigateur)
#    - Utilisez-les pour les configurations publiques uniquement
#    - NE METTEZ JAMAIS de clés API sensibles avec le préfixe VITE_
#
# 2. Variables sans préfixe VITE_ :
#    - Sont uniquement disponibles côté serveur
#    - Utilisez-les pour les clés API et secrets
#
# 3. Sécurité :
#    - Ne commitez JAMAIS ce fichier avec des valeurs réelles
#    - Utilisez .env.local pour vos configurations locales (dans .gitignore)
#    - En production, utilisez des variables d'environnement système ou un gestionnaire de secrets
#
# 4. Format :
#    - Pas d'espaces autour du signe =
#    - Pas de guillemets sauf si la valeur contient des espaces
#    - Les commentaires commencent par #
#
# Exemple CORRECT :
#   OPENAI_API_KEY=sk-1234567890
#
# Exemple INCORRECT :
#   OPENAI_API_KEY = sk-1234567890  (espaces autour du =)
#   OPENAI_API_KEY="sk-1234567890"  (guillemets inutiles)
#
